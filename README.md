# manjaro

----------------------------------------

Для сборки пакетов с AUR в Manjaro


1) Открыть терминал

2) Скопировать и вставить эту команду
```bash
cd ~/ ; git clone https://github.com/linuxshef/manjaro.git ; cd manjaro/scripts ; ./builds
```
